Welcome to RePORTaLiN DocumentationÔÉÅ

RePORTaLiN is a robust data extraction pipeline for processing medical research data from Excel files to JSONL format. It features intelligent table detection, comprehensive logging, progress tracking, and robust error handling.

Quick StartÔÉÅ

Install and run in 3 simple steps:

# 1. Install dependencies
pip install -r requirements.txt

# 2. Run the pipeline
python main.py

# 3. View results in results/dataset/<dataset_name>/

Key FeaturesÔÉÅ

üöÄ Fast & Efficient

Process 43 Excel files in ~15-20 seconds

üìä Smart Table Detection

Automatically splits Excel sheets into multiple tables

üìù Comprehensive Logging

Timestamped logs with detailed operation tracking

üìà Progress Tracking

Real-time progress bars for all operations

üîß Configurable

Centralized configuration management

üìñ Well Documented

Comprehensive user and developer documentation
