RePORTaLiN Documentation
========================

**RePORTaLiN** (Regional Prospective Observational Research for Tuberculosis Analysis in India) is a comprehensive data extraction and harmonization pipeline for tuberculosis research studies. This toolkit enables automated extraction, validation, and de-identification of clinical trial data from PDF forms and Excel datasets.
